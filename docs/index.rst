nlpatl
======

`nlpgatl` is a library for active learning in machine learning experiments.
The goal of NLPatl is assisting user to build high quality labeled dataset. It built on top of transformers, scikit-learn and other machine learning package. It can be applied into both cold start scenario (no any labeled data) and limited labeled data scenario.
